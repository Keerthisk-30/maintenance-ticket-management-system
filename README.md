                                Maintenance Ticket System
Project Overview:

The Maintenance Ticket System is a web-based helpdesk application developed to manage and track maintenance issues efficiently.
It allows users to create maintenance tickets, upload issue attachments, check ticket status, and track updates.
The admin can manage all tickets, monitor ticket analytics, filter tickets, and close resolved issues.

This project was developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.
